# Fiber Command
## API RESTful

    yarn install
    yarn dev

//or

    yarn start


## API endpoints

### Box
|Método|Endpoint|Função|Descrição|
|---|---|---|---|
|POST|/boxes|create|Criar caixa|
|GET|/boxes|find|Buscar caixas|
|GET|/boxes/:id|findOne|Buscar um caixa|
|PUT|/boxes/:id|edit|Editar caixa|
|DELETE|/boxes/:id|remove|Remover caixa|
